Submission Format

STEP 1

Submit link to hosted project in the form given below.

STEP 2

Make a pdf of the project and submit it in the assignment upload section below. Your pdf should contain the following details:

• Project Description: Give a brief about your project description i.e. what was the main aim you were trying to achieve with the project.

• Approach: Write a short paragraph about your approach towards the project and how you have executed it.

• Tech-Stack Used: Mention the Tech Stack which you have used (e.g. HTML, CSS, Node.js etc) along with the things you have achieved using them e.g. HTML :- Made the structure of the website by identifying the appropriate elements required

• Repository Link: Link to your github repository (should be public) containing your project files

• Website Link: Link to the deployed website
